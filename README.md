# Kai_Bailey_GIS

## GitHub Pages Portfolio

A minimal static portfolio site is included in `index.html` and `css/styles.css` for use with GitHub Pages.

How to deploy on GitHub Pages:

- In your repository settings choose **Pages** and set the source to the `main` branch and the root (`/`) (or the `docs/` folder if you prefer). The `index.html` at repository root will be served.
- Alternatively, enable Pages from `gh-pages` branch if you want to publish from a branch.

Editing projects:

- Open `index.html` and duplicate one of the `<article class="card">` blocks in the `.projects` section for each project.
- Update the title, description and the `href` on the `a.card-link` element to point to a project page or external link.
