Unofficial County of Los Angeles Infastructure (ArcGIS Dashboard)

Interactive dashboard exploring various infrastructure projects of Los Angeles. I created this using ArcGIS Dashboards so that I could practice my ArcGIS Online skills and get familiar with the variety of apps ESRI offers. This map was heavily inspired by LA County Public Works' Infrastructure Dashboard, which is linked on the splash page when you open the dashboard.

Seattle Parks — Interactive Trail Map

Seattle Parks & Recreation launched an interactive trail map to enhance public park access. My responsibilities included data editing using LiDAR, historical records, and aerial imagery. This project took months to complete since I had to manually edit over 200+ miles of park trails throughout Seattle. Read the announcement and open the live map below.

Duwamish Valley Project Map

Interactive map highlighting the Duwamish Valley Project area, community features, and planning resources. My responsibilities included leading meetings with department heads, polygon maintenance, symbology, and data editing. This map was the first of its kind for the City of Seattle, enhancing data discoverability through a centralized web map with data from multiple departments.

Seattle Museum Finder

An interactive museum finder for Seattle — search museums, view locations on a map, and access details for planning visits. This project was created as part of a final for a Web GIS course at the University of Washington. My responsibilities included the Isochrone distance, basemap, and routing directions API. This project taught me about Github Collaboration, HTML, CSS, and Javascript. It also taught me the skills to make this portfolio!
